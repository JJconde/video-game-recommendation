Problem:

So many things to think about...

Cannot search for similar games or similar to 2 or more games.


How to Use:

You can search for multiple games by typing something similar to the right.

You can also look for games similar to one game, but not similar to another.


Examples

Search 1: "Call of Duty"

Search 2: "Half-Life" Portal

Search 3: halflife -doom

Model Used: Doc2Vec

The model was built by running the algorithm through the sentences iterator twice: once to build the vocab, and once to train the model on the input data

The model learns a vector representation for each word and for each label in the dataset.


Similarity

The model searches for the meaning of the label, in this case video game titles. It then searches for the closes vector representation of each game, and return those games.

Using vectors, we can easily add or subtract another game, and get the most similar games of the result.
